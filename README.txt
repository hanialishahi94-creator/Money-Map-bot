راهنمای ساده نصب و دیپلوی - پنل ادمین بات مانی‌مپ
====================================================

این پکیج شامل بات تلگرام (که قبلاً داشتی، با کمی اصلاح) + یک پنل ادمین وب جداست.

فایل‌های مهم:
- telegram_bot.py  → همان بات قبلی توست، فقط الان دیتای کاربران/VIP/تحلیل‌ها رو
                     از یک دیتابیس می‌خوند، نه از حافظه موقت. (بدون این کار،
                     دیتا با هر بار ری‌استارت شدن سرویس از بین می‌رفت)
- admin_panel.py   → خود پنل ادمین (یک وب‌سایت ساده)
- db.py            → دیتابیس مشترک بین بات و پنل (فایل moneymap.db)
- main.py          → فایلی که هم بات و هم پنل رو با هم روشن می‌کنه
- templates/       → ظاهر صفحات پنل
- requirements.txt → لیست پکیج‌های لازم پایتون
- Procfile         → برای دیپلوی روی Railway

نحوه دیپلوی روی Railway (همون جایی که الان باتت رو می‌چرخونی):
1. این پروژه رو جای پروژه قبلی بات بگذار (یا یک ریپازیتوری گیت‌هاب جدید بساز و
   این فایل‌ها رو توش بریز و به Railway وصلش کن).
2. در تنظیمات Railway -> Variables، این متغیرها رو ست کن:
   - BOT_TOKEN  → همون توکن باتت
   - VIP_CHANNEL_LINK → لینک کانال VIP (اگه قبلاً ست نکردی)
   - ADMIN_USERNAME → نام کاربری دلخواه برای ورود به پنل (اختیاری)
   - ADMIN_PASSWORD → رمز عبور دلخواه برای ورود به پنل (اختیاری)
   - SECRET_KEY → یک متن دلخواه و طولانی برای امنیت بیشتر (اختیاری)
3. مهم‌ترین نکته: Railway به‌صورت پیش‌فرض فایل‌ها رو با هر دیپلوی جدید پاک می‌کنه.
   برای اینکه دیتای کاربرات (اسم، شماره، VIP) از بین نره، باید یک "Volume"
   (فضای ذخیره دائمی) به سرویس اضافه کنی و آن را روی مسیر  /data  وصل کنی.
   در Railway: روی سرویس کلیک کن -> تب Volumes -> Add Volume -> Mount Path را
   /data بگذار. بعدش دیتابیس به‌صورت خودکار همون‌جا ذخیره می‌شه و با هر
   آپدیت کد، دیتا باقی می‌مونه.
4. روی Deploy بزن. وقتی سرویس بالا اومد:
   - بات تلگرام مثل قبل کار می‌کنه.
   - پنل ادمین روی همون آدرسی که Railway بهت می‌ده در دسترسه (مثلاً
     something.up.railway.app). برای دیدنش کافیه همون آدرس رو در مرورگر باز
     کنی و وارد بشی.

اگه نخوای الان Volume بسازی، همه‌چیز کار می‌کنه ولی با هر دیپلوی جدید (نه هر
ری‌استارت ساده، فقط وقتی کد رو آپدیت می‌کنی) لیست کاربرا/VIP از نو خالی می‌شه.

نکته درباره تحلیل هفتگی:
به جای اینکه قبلاً مجبور بودی متن تحلیل‌ها رو داخل کد بات تغییر بدی و دوباره
دیپلوی کنی، الان از منوی "تحلیل هفتگی" در پنل ادمین، متن و تاریخ هرکدوم از سه
تحلیل (طلا، دلار، بیتکوین) رو می‌نویسی و دکمه ذخیره رو می‌زنی. همون لحظه توی
بات هم همون متن نمایش داده می‌شه؛ بدون نیاز به دیپلوی دوباره.
